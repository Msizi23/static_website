# Accessibility Portfolio Webpage

**Project Description**
This project focuses on web accessibility and semantic HTML by creating a personal portfolio webpage. The webpage presents my professional background, skills, experience, projects, and contact information in a structured and accessible format.

**Purpose**
The purpose of this project is to practice accessible web development and demonstrate how proper HTML structure and accessibility techniques can improve usability and user experience for all users.

**Features:**
* Semantic HTML structure using elements such as header, nav, main, section, article, and footer.
* Accessible navigation with ARIA labels.
* Responsive layout for different screen sizes.
* Accessible contact form with labeled inputs.
* Structured presentation of professional experience, skills, and achievements.
